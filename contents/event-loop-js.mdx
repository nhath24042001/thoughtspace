## The Event Loop Overview

The event loop processes tasks from:

- **Call Stack**
- **Macro Task Queue (setTimeout, setInterval)**
- **Micro Task Queue (Promises, `queueMicrotask`)**

### Example

```js
console.log('1');

setTimeout(() => console.log('2'), 0);

Promise.resolve().then(() => console.log('3'));

console.log('4');
```
