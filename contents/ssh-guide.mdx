## What is SSH?

SSH (Secure Shell) is a protocol that allows encrypted communication between two systems over an insecure network.

### Common Use Cases

- Remote login to a server
- Git operations with SSH keys
- Secure file transfer (SCP, SFTP)

### Generate SSH Key

```bash
ssh-keygen -t ed25519 -C "your_email@example.com"
```
