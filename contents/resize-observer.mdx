## What is ResizeObserver?

`ResizeObserver` is a browser API that allows you to track changes in the size of an element.

### Why use it?

- No need to poll dimensions on every frame.
- Works with flex, grid, and dynamic layout changes.
- Efficient for responsive components.

```js
const observer = new ResizeObserver((entries) => {
  for (const entry of entries) {
    console.log('Size changed:', entry.contentRect);
  }
});

observer.observe(document.querySelector('#my-element'));
```
